============
Installation
============

Requirements
===========

* TYPO3 v11.5, v12.4 or v13.4
* PHP 7.4 or higher

Installation via Composer
=======================

The recommended way to install this extension is using Composer:

.. code-block:: bash

   composer require mst/mst-yaml2tca

Installation from TYPO3 Extension Repository
=========================================

You can also install the extension from the TYPO3 Extension Repository (TER):

1. Log into the TYPO3 backend
2. Go to **Admin Tools > Extensions**
3. Click on "Get Extensions"
4. Search for "mst_yaml2tca"
5. Click "Import and Install"
